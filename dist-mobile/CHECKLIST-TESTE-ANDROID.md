# Checklist de teste Android - Nova Forma CRM

Use este roteiro depois de instalar `app-debug.apk` no celular.

## Instalacao

- [ ] Se for via USB, `VERIFICAR-CELULAR.bat` mostra o aparelho como `device`.
- [ ] Se for via Wi-Fi, `PAREAR-WIFI.bat` pareou com codigo.
- [ ] Se for via Wi-Fi, `CONECTAR-WIFI.bat` conectou no IP:PORTA do celular.
- [ ] `INSTALAR-VIA-USB.bat` ou `INSTALAR-VIA-WIFI.bat` instalou o APK com sucesso.

## Teste online

- [ ] Abrir app.
- [ ] Conferir icone, nome e splash.
- [ ] Confirmar que o app nao parece uma pagina web solta dentro do WebView.
- [ ] Fazer login.
- [ ] Abrir Dashboard.
- [ ] Conferir navegacao inferior.
- [ ] Abrir Leads.
- [ ] Criar lead de teste.
- [ ] Abrir detalhe do lead.
- [ ] Testar botao WhatsApp.
- [ ] Testar copiar mensagem/template.
- [ ] Abrir Pipeline.
- [ ] Alterar status do lead.
- [ ] Abrir Tarefas.
- [ ] Criar tarefa.
- [ ] Concluir tarefa.
- [ ] Abrir Templates.
- [ ] Abrir Importar com IA.
- [ ] Abrir Configuracoes.
- [ ] Ver secao de sincronizacao.
- [ ] Testar logout.
- [ ] Entrar novamente.

## Teste offline

- [ ] Com login feito, abrir Dashboard, Leads, Tarefas e Templates para cachear dados.
- [ ] Ativar modo aviao.
- [ ] Fechar e abrir o app.
- [ ] Confirmar que o app abre offline.
- [ ] Confirmar que aparece indicador offline.
- [ ] Criar lead offline.
- [ ] Editar lead offline.
- [ ] Criar tarefa offline.
- [ ] Registrar interacao offline.
- [ ] Preparar/copiar mensagem de WhatsApp.
- [ ] Abrir Configuracoes e ver operacoes pendentes.
- [ ] Desativar modo aviao.
- [ ] Sincronizar.
- [ ] Confirmar se o lead aparece no CRM web.
- [ ] Confirmar que a fila ficou limpa.
- [ ] Apagar lead de teste depois.

## UX mobile

- [ ] Botoes estao grandes o suficiente.
- [ ] Textos estao legiveis.
- [ ] Menu inferior nao cobre conteudo importante.
- [ ] Campos nao ficam escondidos pelo teclado.
- [ ] Pagina do lead esta confortavel.
- [ ] Pipeline em chips/tabs esta facil de usar.
- [ ] Cards de leads estao compactos.
- [ ] Templates estao faceis de copiar/usar.
- [ ] Dashboard esta limpo.
- [ ] Status offline esta claro.
- [ ] Nao ha overflow horizontal.
- [ ] WhatsApp abre corretamente.
